# Space Hack

I den här övningen får du träna på att kombinera Flexbox och CSS Grid i samma layout. Tanken är att du steg för steg ska fundera på vilka delar av designen som passar bäst för Grid och vilka som lämpar sig för Flexbox.

[Figmaskiss](https://www.figma.com/design/oHzAfe4fp1kqtnQiZwTrbY/Space-hack?node-id=184-124&t=M1qKS8i6PKXKLu40-0)

Titta noga på strukturen i skissen och försök först identifiera vilka element som behöver placeras i ett rutnät och vilka som behöver kunna justeras i rad eller kolumn. När du känner att du har en idé kan du börja bygga upp layouten i HTML och styra placeringen med CSS.
